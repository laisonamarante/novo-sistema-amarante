# PROJETO AMARA: GESTÃO E EXPANSAO AMARANTE
# Status: Observação Ativa (Ghost Mode)

## 1. Conectividade e Observabilidade
- **Amara Email (Postgres)**: Monitoramento de fluxo de entrada, categorização de urgência e volume.
- **Amarante Chat (SQLite)**: Indexação de conversas para aprendizado de perfil de cliente e sugestão de fluxos.
- **Novo Amarante (MySQL)**: Auditoria de transações, cadastro de processos e velocidade operacional.

## 2. Governança e Inteligência
- **Biblioteca de Conhecimento**: Indexação automática de manuais (BB, regras de negócios).
- **Protocolo de Segurança**: Acesso leitura-apenas. Zero resposta automatizada para o cliente final.
- **Insights**: Identificação automática de padrões (quais processos travam, quais atendentes precisam de suporte).

## 3. Rotina de Gerenciamento
- **Diário (08:00h)**: Resumo Executivo (Estabilidade do sistema + KPI de atendimento).
- **Semanal**: Relatório de "Gargalos Operacionais" (onde o sistema está lento).
- **Quinzenal**: Proposta de Projeto de Expansão (ex: novos produtos, automações de preenchimento).
